# PrintBytes Admin Dashboard

A modern admin dashboard for a local printing shop to manage daily sales, expenses, and net profit. Features interactive charts comparing monthly performance and a responsive interface.

## Features
- Input daily sales, expenses, and partitions
- Dashboard showing total overall sales, monthly sales, expenses, and net profit
- Interactive charts powered by Chart.js comparing current and previous months
- User-friendly and responsive interface

## Tech Stack
- Laravel
- TailwindCSS
- Chart.js
- JavaScript

## Usage
Log in to the dashboard to manage sales and expenses

View charts to track business performance
